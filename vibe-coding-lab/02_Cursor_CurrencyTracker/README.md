# Live Exchange Rate CLI

A simple terminal-based application that fetches real-time exchange rates using a public API.

This project demonstrates how to integrate an external API and display live currency data directly in the command line interface (CLI). It is designed as a lightweight experiment in building API-driven terminal tools.

## Features

- Fetches live exchange rates from an external API
- Displays currency rates directly in the terminal
- Simple and lightweight CLI interface
- Easy to extend with additional currencies or features
